# Team Directory

This directory contains profile files for team members. When a new developer runs the environment verification script (`npm run verify`) and their environment is correctly configured, they'll see a "Meet the Team" section introducing everyone.

## Adding Your Profile

1. Copy `TEMPLATE.txt` to a new file named `your-name.txt` (e.g., `jane-doe.txt`)
2. Fill in your details:
   - **name**: Your full name
   - **location**: Your floor and seat (e.g., "Floor 3, Seat 42")
   - **cats**: Number of cats you own (very important data)
3. Commit your profile to the repo

## Example

```
# Team Member Profile

name: Jane Doe
location: Floor 2, Seat 15
cats: 3
```

## Why Cats?

Because every good team needs to know the critical cat statistics.
